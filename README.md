使用 app.asar 文件替换 Xmind 安装路径中的 resources 文件夹中的 app.asar 文件即可。注意禁止 xmind 自动更新，发送程序错误日志。

Windows 11上安装后的目录默认在C:\Users\[登录用户名]\AppData\Local\Programs\Xmind
